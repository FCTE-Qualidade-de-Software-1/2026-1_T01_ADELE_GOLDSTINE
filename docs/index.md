# Grupo Adele Goldstine — Qualidade de Software 2026.1

## Sobre o Projeto

Este projeto realiza a **avaliação da qualidade do [SuaGradeUnB](https://suagradeunb.com.br/)**, uma aplicação web *open source* para gerenciamento de grades de horários de estudantes da Universidade de Brasília, desenvolvida pelo grupo `unb-mds` ([repositório](https://github.com/unb-mds/2023-2-SuaGradeUnB)).

A análise utiliza o modelo **ISO/IEC 25010**, com foco nas características de **manutenibilidade** e **segurança**, a fim de identificar pontos fortes e oportunidades de melhoria no sistema.

## Adele Goldstine

Adele Goldstine (1920–1964) foi uma matemática e cientista da computação americana, autora do primeiro manual técnico do ENIAC e responsável por treinar suas seis programadoras originais. Seu trabalho documentando e ensinando o uso da máquina foi peça-chave para tornar a computação programável uma prática viável.
